孫 志貴｜山東出身｜Dorico だけで楽譜浄書＆劇伴作曲｜東京で作曲勉強中｜日英中三つ言語話せる｜Mac使い。

  • 🎓 レスブリッジ大学音楽学部出身で、デジタル音響芸術専攻卒業（学士号）。
  • 🏫 東京で音大の大学院を申し込み中。

平成２年（中国）山東生まれ。平成30年（カナダの）レスブリッジ大学でデジタル音響芸術専攻修了し音楽学士号を獲得。平成31年春から令和２年11月迄東京の専門学校で映像作曲を学び続いていた。作曲を Arlan N. Schultz に師事、主には Steinberg Dorico で譜面作曲。

🎼 デモ作品は SoundCloud へ：https://soundcloud.com/shikisuen/sets/spotlighted-portfolios

譜面づきの一部の曲：《琴で語る、その剣の序づ》《運命の呼び掛け》《天機の変》《譜面で勝つ》《雲の温もり》《雲の寒さ》《雲の侘び》《仇討ちの果》

  • 🎼 楽譜浄書について、使えるのは Sibelius と Dorico だけで、今は Dorico だけ使っています。Dorico は Finale の何倍より効率的な譜面浄書ソフトで、素早く出版レベルの総譜とパート譜を作れます♪ 。
  • 💻 DTM用パソコンのシステムを保守することは上手く出来ます（mac か Windows か問わず）。
    Linux / FreeBSD に基づく VPS サーバーで HTML ウェブサイトを築ることができます。
    PHP+MySQL 環境での一部ウェブサイトソフト（XenForo / PHPBB など）のインストール＆配置も出来ます。
